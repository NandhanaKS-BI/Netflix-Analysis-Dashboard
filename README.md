# 📊 Netflix Content Analysis Dashboard (Tableau)

## 📌 Project Overview
This project analyzes Netflix content distribution based on genre, release year, country, and content type using Tableau.  
The dashboard provides insights into trends in movies and TV shows available on Netflix.

---

## 🎯 Objective
The objective of this project is to:
- Analyze Netflix content distribution by genre
- Examine content growth over the years
- Compare Movies vs TV Shows
- Identify top contributing countries
- Understand content trends and patterns

---

## 🛠 Tools & Technologies Used
- Tableau
- Tableau Public (for publishing)
- Excel / CSV Dataset
- Data Cleaning & Preparation

---

## 📊 Key Metrics Included
- Total Titles
- Total Movies
- Total TV Shows
- Content by Genre
- Content by Release Year
- Content by Country

---

## 📈 Dashboard Analysis Includes
- Genre-wise content distribution
- Year-wise content growth trend
- Movie vs TV Show comparison
- Country-wise content contribution
- Rating distribution analysis (if included)

---

## 📺 Dashboard Features
- Interactive filters (Genre, Year, Country, Type)
- Dynamic visualizations
- Bar charts and line charts
- Clean and user-friendly layout
- Drill-down analysis
